# Machine Learning Models List

### ProsusAI/finbert
**Desc:** FinBERT is a pre-trained NLP model to analyze sentiment of financial text. It is built by further training the BERT language model in the finance domain, using a large financial corpus and thereby fine-tuning it for financial sentiment classification.  
**Docs:** https://huggingface.co/ProsusAI/finbert

### yiyanghkust/finbert-tone
**Desc:** FinBERT is a BERT model pre-trained on financial communication text. The purpose is to enhance financial NLP research and practice. It is trained on the following three financial communication corpus. The total corpora size is 4.9B tokens.  
**Docs:** https://huggingface.co/yiyanghkust/finbert-tone

### tabularisai/ModernFinBERT
**Desc:** A fine-tuned financial sentiment analysis model based on ModernBERT, trained on synthetic and real financial data cleaned through an automated AI agentic pipeline.  
**Docs:** https://huggingface.co/tabularisai/ModernFinBERT

### FinGPT
**Desc:** FinGPT is deeply committed to fostering an open-source ecosystem dedicated to Financial Large Language Models (FinLLMs). FinGPT envisions democratizing access to both financial data and FinLLMs.  
**Docs:** https://huggingface.co/FinGPT

### StephanAkkerman/FinTwitBERT-sentiment
**Desc:** FinTwitBERT-sentiment is a finetuned model for classifying the sentiment of financial tweets. It uses FinTwitBERT as a base model, which has been pre-trained on 10 million financial tweets.  
**Docs:** https://huggingface.co/StephanAkkerman/FinTwitBERT-sentiment

### isolation-forest/setfit-absa-polarity
**Desc:** This is a SetFit model that can be used for Aspect Based Sentiment Analysis (ABSA). This SetFit model uses cointegrated/rubert-tiny2 as the Sentence Transformer embedding model.  
**Docs:** https://huggingface.co/isolation-forest/setfit-absa-polarity

### santhosraj14/sarimax_grach-for-price-forecasting
**Desc:** A hybrid time series model for price forecasting  
**Docs:** https://www.kaggle.com/models/santhosraj14/sarimax_grach-for-price-forecasting

### AdaptLLM/finance-LLM
**Desc:** Inspired by human learning via reading comprehension, we propose a simple method to transform large-scale pre-training corpora into reading comprehension texts, consistently improving prompting performance across tasks in biomedicine, finance, and law domains. Our 7B model competes with much larger domain-specific models like BloombergGPT-50B.  
**Docs:** https://huggingface.co/AdaptLLM/finance-LLM

### curiousily/tiny-crypto-sentiment-analysis
**Desc:** Fine-tuned (with LoRA) version of TinyLlama on cryptocurrency news articles to predict the sentiment and subject of an article. The dataset used for training is Crypto News+.  
**Docs:** https://huggingface.co/curiousily/tiny-crypto-sentiment-analysis

### TimesFM (Time Series Foundation Model)
**Desc:** A pretrained time-series foundation model developed by Google Research for time-series forecasting on various datasets, including financial data.  
**Docs:** https://github.com/google-research/timesfm

### Temporal Fusion Transformer (TFT)
**Desc:** A deep learning architecture for multi-horizon time series forecasting, widely used in financial forecasting and demand prediction.  
**Docs:** https://github.com/google-research/google-research/tree/master/tft

### N-BEATS
**Desc:** Neural Basis Expansion Analysis for Time Series - a pure deep learning approach for forecasting, effective for financial time series.  
**Docs:** https://github.com/ElementAI/N-BEATS

### LightGBM
**Desc:** Gradient boosting framework that uses tree-based learning algorithms, highly efficient for large-scale financial data and feature engineering.  
**Docs:** https://github.com/microsoft/LightGBM

### XGBoost
**Desc:** Extreme Gradient Boosting library, one of the most popular ML algorithms for structured/tabular financial data and trading signals.  
**Docs:** https://github.com/dmlc/xgboost

### CatBoost
**Desc:** Gradient boosting library developed by Yandex, excellent for handling categorical features common in financial datasets.  
**Docs:** https://github.com/catboost/catboost

### Prophet
**Desc:** Forecasting tool by Meta (Facebook) for time series data with strong seasonal effects, suitable for financial market trends.  
**Docs:** https://github.com/facebook/prophet

### AutoGluon-TimeSeries
**Desc:** AutoML framework for time series forecasting with ensemble methods, great for automated trading strategy development.  
**Docs:** https://auto.gluon.ai/stable/tutorials/timeseries/index.html

### FinRL
**Desc:** Deep Reinforcement Learning framework for automated trading, providing implementations of DRL algorithms for quantitative finance.  
**Docs:** https://github.com/AI4Finance-Foundation/FinRL

### Stable-Baselines3
**Desc:** Set of reliable implementations of reinforcement learning algorithms, commonly used for developing trading agents and portfolio management.  
**Docs:** https://github.com/DLR-RM/stable-baselines3

### RLlib (Ray)
**Desc:** Scalable reinforcement learning library for building trading bots and backtesting strategies at scale.  
**Docs:** https://docs.ray.io/en/latest/rllib/index.html

### PyPortfolioOpt
**Desc:** Portfolio optimization library implementing Markowitz, Black-Litterman, and other modern portfolio theory algorithms.  
**Docs:** https://github.com/robertmartin8/PyPortfolioOpt

### Riskfolio-Lib
**Desc:** Python library for portfolio optimization and quantitative strategic asset allocation, implementing various risk measures.  
**Docs:** https://github.com/dcajasn/Riskfolio-Lib

### LSTM (Long Short-Term Memory)
**Desc:** Recurrent neural network architecture widely used for stock price prediction and time series forecasting in finance.  
**Docs:** https://pytorch.org/docs/stable/generated/torch.nn.LSTM.html

### GRU (Gated Recurrent Unit)
**Desc:** Simplified variant of LSTM, efficient for sequence modeling in financial time series prediction.  
**Docs:** https://pytorch.org/docs/stable/generated/torch.nn.GRU.html

### Transformer for Time Series
**Desc:** Attention-based architecture adapted for time series forecasting, capturing long-range dependencies in market data.  
**Docs:** https://huggingface.co/docs/transformers/model_doc/time_series_transformer

### Isolation Forest
**Desc:** Anomaly detection algorithm useful for detecting market anomalies, fraud detection, and outlier detection in trading.  
**Docs:** https://scikit-learn.org/stable/modules/generated/sklearn.ensemble.IsolationForest.html

### Autoencoder for Anomaly Detection
**Desc:** Neural network for unsupervised anomaly detection in financial transactions and unusual market behavior.  
**Docs:** https://keras.io/examples/timeseries/timeseries_anomaly_detection/

### ARIMA/SARIMA
**Desc:** Classical statistical models for time series forecasting, baseline for financial market prediction.  
**Docs:** https://www.statsmodels.org/stable/generated/statsmodels.tsa.arima.model.ARIMA.html

### GARCH Models
**Desc:** Generalized Autoregressive Conditional Heteroskedasticity - modeling and forecasting financial volatility.  
**Docs:** https://arch.readthedocs.io/en/latest/univariate/introduction.html

### VAR (Vector Autoregression)
**Desc:** Multivariate time series model for capturing relationships between multiple financial assets.  
**Docs:** https://www.statsmodels.org/stable/vector_ar.html

### Random Forest
**Desc:** Ensemble learning method for classification and regression, used for feature importance in trading signals.  
**Docs:** https://scikit-learn.org/stable/modules/generated/sklearn.ensemble.RandomForestClassifier.html

### TabNet
**Desc:** Attention-based neural network for tabular data, interpretable and effective for financial data analysis.  
**Docs:** https://github.com/dreamquark-ai/tabnet

### Neural Prophet
**Desc:** Neural network-based time series forecasting inspired by Prophet, with autoregression and deep learning capabilities.  
**Docs:** https://github.com/ourownstory/neural_prophet

### Darts
**Desc:** Python library for time series forecasting with multiple models including statistical and deep learning approaches.  
**Docs:** https://github.com/unit8co/darts

### TSMixer
**Desc:** MLP-based architecture for multivariate time series forecasting, efficient for financial multi-asset prediction.  
**Docs:** https://huggingface.co/docs/transformers/model_doc/patchtsmixer

### Chronos
**Desc:** Pretrained time series forecasting models by Amazon, zero-shot forecasting for various financial instruments.  
**Docs:** https://github.com/amazon-science/chronos-forecasting

### Nixtla TimeGPT
**Desc:** Foundation model for time series forecasting, offering API-based predictions for financial data.  
**Docs:** https://github.com/Nixtla/nixtla

### TensorTrade
**Desc:** Framework for building and training trading agents using deep reinforcement learning.  
**Docs:** https://github.com/tensortrade-org/tensortrade

### Backtrader
**Desc:** Python framework for backtesting trading strategies with event-driven architecture.  
**Docs:** https://github.com/mementum/backtrader

### zipline-reloaded
**Desc:** Pythonic algorithmic trading library for backtesting trading strategies.  
**Docs:** https://github.com/stefan-jansen/zipline-reloaded

### sklearn Ensemble Models
**Desc:** Collection of ensemble methods (AdaBoost, GradientBoosting, etc.) for trading signal generation.  
**Docs:** https://scikit-learn.org/stable/modules/ensemble.html

### DeepAR
**Desc:** Probabilistic forecasting with autoregressive recurrent networks, developed by Amazon for time series.  
**Docs:** https://docs.aws.amazon.com/forecast/latest/dg/aws-forecast-recipe-deeparplus.html

### TCN (Temporal Convolutional Network)
**Desc:** Convolutional approach to sequence modeling, effective for financial time series with long sequences.  
**Docs:** https://github.com/locuslab/TCN

### WaveNet
**Desc:** Deep generative model for time series, originally for audio but adapted for financial forecasting.  
**Docs:** https://github.com/ibab/tensorflow-wavenet

### SHAP (SHapley Additive exPlanations)
**Desc:** Model interpretability framework for explaining predictions of trading models and feature importance.  
**Docs:** https://github.com/shap/shap

### LIME (Local Interpretable Model-agnostic Explanations)
**Desc:** Explains predictions of ML models, useful for understanding trading decisions.  
**Docs:** https://github.com/marcotcr/lime
